# schwarm

**Multi-agent orchestration plugin for Claude Code.**

Parallel development streams. Multi-model deliberation. Swarm intelligence for your codebase.

> 🐝 Coming soon. Follow [@bitflorist](https://github.com/bitflorist) for updates.

## What it will do

- Run multiple development streams in parallel (backend, frontend, mobile, data)
- Multi-model chat: Claude + Codex + Gemini debate with you in the loop
- GitHub Issues as single source of truth
- GSD-compatible — works alongside Get Shit Done framework

## Install (preview)

```bash
claude --plugin-dir ./schwarm
```
